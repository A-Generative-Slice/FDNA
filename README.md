# Najeema Afrin Fashion Designer Portfolio

A premium, elegant portfolio website designed for fashion designer **Najeema Afrin**.

## Features
- **Modern Aesthetic**: Maroon, White, and Beige color palette.
- **Responsive Design**: Works perfectly on mobile and desktop.
- **CV Export**: Press `Ctrl + P` to export the entire portfolio as a professional A4 CV.
- **Individual Portfolio Export**: Export specific portfolio pieces one by one.
- **SEO Optimized**: Semantic HTML and meta tags included.

## How to Deploy to GitHub Pages
1. Create a new repository on GitHub named `FDNA` (or any name you prefer).
2. Upload these files (`index.html`, `style.css`, `script.js`) to the repository.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment** > **Branch**, select `main` (or the branch you uploaded to) and `/ (root)`.
5. Click **Save**. Your website will be live at `https://your-username.github.io/FDNA/`.

## Customization
- **Images**: Replace the placeholder text in `index.html` within the `<div class="portfolio-img">` tags with `<img>` tags once you have your actual work images.
- **Personal Info**: Update the placeholders like `[Your Phone Number]`, `[Your Email ID]`, and `[Your DOB]` in the `index.html` file.
